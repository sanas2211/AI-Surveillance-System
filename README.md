# 🔍 AI Surveillance System (YOLOv8)

An AI-powered surveillance system built using YOLOv8 and Streamlit for real-time monitoring and security analysis.

---

## 🚀 Features
- 🎯 Real-time Object Detection (YOLOv8)
- 👤 Face Recognition System
- 🚨 Intrusion Detection Zones
- 🎥 Webcam & Video Monitoring
- ⚠️ Alert System (Person / Dangerous Objects)
- 📊 Live Dashboard UI
- 📝 Log Saving (CSV)

---

## 🛠️ Tech Stack
- Python
- YOLOv8 (Ultralytics)
- OpenCV
- Streamlit
- Face Recognition (dlib)
